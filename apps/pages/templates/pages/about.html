About

What is Smart Hub?

Smart Hub is an open source application designed to allow Internet of Things (IoT) devices to communicate with each other, with Smart Hub acting as the integration platform.

Can I contribute to it's development?

Yes! All contributions are appreciated, you can view the code and find out more at the projectGitHub Repo.

What protocols does it support?

At present, it supports the following protocols:

  • Zigbee
Given the open source development, it is expected that this list will continue to grow overtime. If the protocol you need is not supported, you are welcome to develop it and create pull request on GitHub - or let us know and we'll priortise based on demand.

How does it work?

Zigbee

For Zigbee communication to work, a MQTT broker must be setup to communicate with the Zigbee devices. Smart Hub can then be configured to connect to the MQTT broker with the relevant device topics (i.e. the channel each device is publishing to). With these configurations in place, Smart Hub will then facilitate all communication to and from the devices - based on your personal configurations.

What can Smart Hub do?

Good question!

  • Secure your home smart devices and automations with an authentication system
  • Create automations using event triggers and responses
  • View device communication logs
  • Export logs in CSV for data analysis
  • Receive notifications when events are triggered, on your mobile device/tablet/computer/email/etc.
  • Create states for your devices that can be toggled, e.g. turning devices on and off, or setting RGB light bulb values.
  • Group devices by geographic location - in future you will be able to create location events and/or toggle location devices
  • Store your home GPS location - in future you will be able to use this to create automations, when you leave or get close to home.
