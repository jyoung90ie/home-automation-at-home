Create Device State

A State can be thought of as the exact configuration required for the device to function as you would like it when an event is triggered, or a particular event is triggered. An example is a generic light bulb, that can be switched on or off - this means the light bulb has two states: on, and, off. But what if you have an LED bulb that can change colours? Well now you have many state options, you may specify the RGB (Red Green Blue) value for each on state.
